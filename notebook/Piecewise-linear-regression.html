Introduction¶

This notebook is a simple Python demonstration of piecewise linear regression. Piecewise linear regression can be used to represent the simplified relationship between outdoor air temperature and whole building energy consumption, and this simple data-driven model of a whole building energy consumption has been widely used for retrofitting application [1,2,3]. We provide a reproducible Python notebook for a piecewise linear regression model. First, we generate a toy dataset from the known piecewise linear regression model. Then, we provide an optimization based approach to estimate model parameters of the piecewise linear regression mode

Piecewise linear model¶

To generate a toy dataset, we assume we know the true model parameters of a piecewise linear model. The true model is a combination of two linear lines intersecting at $x=10$:

$$ \begin{align} y &= -x+5 &\text{ if } &x<10 \\ y &= -x+5+3(x-10) &\text{ if } &x\geq10. \end{align} $$

This can be a written in this format:

$$y=5-x+3(x-10)^{+},$$

where $()^{+}$ is $0$ when the value in $()$ is less than $0$. Likewise, $()^{-}$ is $0$ when the value in $()$ is greater than $0$.

In real world, we don't have 100% accurate measurement. For the simplicity, all sources of uncertainties (e.g., measurement errors, noise in data generation process, etc.) are modeled as a single noise $\varepsilon$. We assume this noise follows a normal distribution of $\varepsilon \sim \text{N}(\mu_{\varepsilon}=0,\sigma_{\varepsilon}=2)$.

With data specific index $i$:

$$y_{i}=5-x_{i}+3(x_{i}-10)^{+}+\varepsilon_{i}.$$

from scipy import optimize

def piecewise_linear(x, x0, b0, b1, b2):
    condlist = [x < x0,  x >= x0]
    funclist = [lambda x: b0+b1*x, lambda x: b0+b1*x+b2*(x-x0)]
    return np.piecewise(x, condlist, funclist)

p , e = optimize.curve_fit(piecewise_linear, x_data, y_data) #least square
